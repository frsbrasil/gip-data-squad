# Output Quality Checklist — GIP Data Analyst Squad

## Data Quality
- [ ] Data source identified and documented
- [ ] Row counts validated before and after cleaning
- [ ] Nulls and duplicates handled with rationale
- [ ] Date ranges confirmed

## Analysis Quality
- [ ] Business question explicitly answered
- [ ] All claims traceable to data
- [ ] Statistical tests appropriate for data type and sample size
- [ ] Limitations stated

## Chart Quality (SWD)
- [ ] Every chart has an action title
- [ ] Legend removed where direct labels possible
- [ ] Only key data highlighted
- [ ] One message per chart

## Report Quality
- [ ] Executive summary leads with the answer
- [ ] Recommendations have owners and success metrics
- [ ] Numbers have context
- [ ] No unexplained jargon
