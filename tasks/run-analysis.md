# Task: Run Full Analysis Pipeline

## Trigger
User has a dataset and business question and wants end-to-end analysis.

## Phases
1. **Frame** — confirm question, identify data, define success
2. **Ingest** — delegate to ingestion-agent
3. **Analyze** — delegate to eda-agent then stats-agent
4. **Report** — delegate to reporting-agent

## Output
Final report in Markdown (default) or .docx if requested.
