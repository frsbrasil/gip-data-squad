# GIP Data Analyst Squad

5 specialist AI agents for end-to-end data analysis — ETL, EDA, statistics, visualization, and reporting.

Inspired by [`ai-analyst-lab/ai-analyst`](https://github.com/ai-analyst-lab/ai-analyst) and [`HungHsunHan/claude-code-data-science-team`](https://github.com/HungHsunHan/claude-code-data-science-team).

## Agents (5)

| Agent | Role |
|-------|------|
| **data-director** | Chief Orchestrator — routes, plans, phases |
| **ingestion-agent** | ETL, SQL, CSV/Excel, data cleaning |
| **eda-agent** | EDA, profiling, correlations, charts |
| **stats-agent** | Hypothesis tests, regression, ML, forecasting |
| **reporting-agent** | KPI decks, exec summaries, Word/PPT |

## Quick Start

```
@data-director     # Full pipeline orchestrator
*run-analysis      # Frame → Analyze → Report
*quick-explore     # Fast EDA
*diagnose          # Triage a data problem
```
